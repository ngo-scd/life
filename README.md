# 🏛️ ГО «Суспільний центр дії» (Public Action Center)

![Status](https://img.shields.io/badge/Status-Active-brightgreen)
![Version](https://img.shields.io/badge/Version-1.2.0-blue)
![Tech](https://img.shields.io/badge/Stack-Tailwind_CSS_%7C_HTML5_%7C_JS-orange)

Сучасна веб-платформа громадської організації, створена для системної боротьби з корупцією, захисту прав громадян та розбудови прозорого суспільства в Україні.

## 🚀 Особливості проекту

- **Premium UI/UX:** Сучасний темний інтерфейс з акцентним помаранчевим кольором (`#FF6433`).
- **HR-Hub:** Інтегрована система пошуку спеціалістів та волонтерів.
- **Fin-Module:** Зручна система копіювання реквізитів (IBAN, Card, Crypto) в один клік.
- **Mobile First:** Повна адаптивність під мобільні пристрої з унікальним "скляним" меню.
- **High Performance:** Використання Tailwind CSS для миттєвого завантаження стилів та AOS для плавної анімації.

## 🛠 Технологічний стек

* **Frontend:** HTML5, CSS3 (Tailwind CSS 3.x)
* **Interactivity:** Vanilla JavaScript (ES6+)
* **Icons:** Phosphor Icons (Fill, Duotone, Bold)
* **Animations:** AOS (Animate On Scroll)
* **Typography:** Inter Google Font

## 📂 Структура секцій

1.  **Hero Section:** Головний меседж та швидкий доступ до допомоги.
2.  **Committees:** Напрямки діяльності (Антикорупція, Медицина, Право).
3.  **Vision & Roadmap:** Стратегічний план розвитку організації.
4.  **Governance:** Прозорий опис структури управління.
5.  **Donate & HR Hub:** Секція для фінансової підтримки та заповнення анкети волонтера.
